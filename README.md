# 🎯 Desafio Criativo — Extraindo Insights do Feedback de Clientes Bancários

Repositório criado para entrega do Desafio Criativo da [DIO](https://www.dio.me/).

---

## 📌 Sobre o Projeto

Este projeto apresenta um **prompt de IA** desenvolvido para analisar feedbacks de clientes bancários e extrair insights acionáveis sobre canais digitais como aplicativo móvel, Pix, cartão de crédito e atendimento por chatbot.

O objetivo é transformar comentários dispersos em informações claras e organizadas para apoiar decisões das equipes de Experiência do Cliente (CX) e Produtos Digitais.

## 🧱 Como o Prompt foi Construído

O prompt foi desenvolvido em 3 etapas:

**Passo 1 — Intenção**
Definição do que a IA deve produzir, para quem e com qual finalidade.

**Passo 2 — Contexto e Restrições**
Descrição dos dados disponíveis, critérios de análise e cuidados com privacidade e integridade das informações.

**Passo 3 — Prompt Final Consolidado**
União de todos os elementos em um único comando claro, com papel da IA, objetivo, formato de saída e restrições obrigatórias.
